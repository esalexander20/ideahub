# IdeaHub: Project Plan

## Phase 1: Core Features (Day 1–2)

- User authentication (login/signup)
- Profile creation and management
- Idea submission form
- Basic idea listing (feed)

## Phase 2: Validation & Collaboration (Day 3–4)

- Upvote/downvote system
- Commenting and feedback
- Idea search and filtering

## Phase 3: Polish & Deployment (Day 5)

- UI/UX improvements
- Bug fixes
- Deploy to Vercel/Netlify

## Stretch Goals

- Real-time collaboration (e.g. live editing)
- AI-powered idea suggestions
- Mobile app version
